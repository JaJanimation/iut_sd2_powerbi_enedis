# IUT SD2 – Power BI Reporting ENEDIS
#### Projet fais par : Jasmine Pioct, Sabrina Moufok et Hadjer Merabet
## Contexte
Projet académique Power BI réalisé dans le cadre du parcours SD2.  
Client fictif : **GreenTech Solutions**, missionnée par **Enedis** pour établir un état des lieux de la performance énergétique des logements d’une ville française à partir des données DPE.

Objectif central : concevoir une application Power BI exploitable, maintenable et documentée, orientée analyse décisionnelle.

## Périmètre du projet
- Analyse de la performance énergétique des logements (neufs et existants)
- Exploitation de données publiques (ADEME, BAN)
- Restitution via tableaux de bord interactifs Power BI
- Déploiement sur Power BI Service

## Livrables
- Application Power BI (.pbix)
- Maquette UX/UI (Figma / Canva)
- Documentation technique (≤ 2 pages)
- Documentation fonctionnelle (≤ 2 pages)
- Vidéo de démonstration (≤ 5 min)

## Fonctionnalités implémentées
### Données
- Import CSV / API
- Nettoyage avancé avec Power Query
- Gestion des erreurs, valeurs manquantes et doublons
- Modélisation en schéma simple

### Modélisation & calculs
- Tables de faits et dimensions (département, commune)
- Mesures DAX simples et avancées
- Optimisation du modèle

### Visualisation
- Indicateurs clés de performance énergétique
- Graphiques comparatifs
- Carte interactive
- Filtres
- Navigation multi-pages avec boutons personnalisés

### Sécurité & performance 
- Diagnostic via l’Analyseur de performances Power BI
